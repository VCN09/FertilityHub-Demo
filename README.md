## 🎯 Visão Geral

**FertilityHub** é uma plataforma completa de gestão clínica para medicina reprodutiva, desenvolvida com arquitetura modular e padrões enterprise-grade.

**Tech Stack:** Python 3.10 + Flask 2.3 + SQLAlchemy + REST APIs + RBAC

## ✨ Características Técnicas

- Backend robusto com Factory Pattern
- ORM moderno com SQLAlchemy
- API REST com JWT authentication
- RBAC completo (4 perfis clínicos)
- Dashboard responsivo
- Deployment pronto (Docker)
